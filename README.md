# 🧠 SAE-SCD — Fuzzy Classifier System (Python)

SAE-SCD is a Fuzzy Classifier System implemented in Python for intelligent
classification and adaptive decision-making. The project provides an
architecture based on fuzzy inference to model uncertainty and classify
system states dynamically.

This repository implements a structured fuzzy classifier including
membership functions, rule base, inference engine, and decision output.

## 📖 Description

SAE-SCD implements a fuzzy classifier architecture capable of evaluating
input variables and producing adaptive decisions using fuzzy rules.

The system models uncertainty using linguistic variables and membership
functions. Based on rule evaluation, the classifier determines output
categories that represent system behavior.

The architecture includes:

- Input variable definition
- Membership functions
- Fuzzy rule base
- Inference engine
- Defuzzification module
- Decision output

---

## 🧠 Fuzzy Classifier Architecture

The classifier follows a classical fuzzy pipeline:

Input Variables → Fuzzification → Rule Base → Inference → Defuzzification → Output

The system allows classification using linguistic variables such as:

- Low
- Medium
- High
- Critical
